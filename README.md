# Bharat Khanna

Website source for [bharatkhanna.dev](https://bharatkhanna.dev) — a Hugo + Netlify site focused on AI systems, agent architectures, retrieval, evaluation, and production engineering.

## Focus Areas

- Agent architectures and orchestration
- LLM evaluation and runtime reliability
- Vector retrieval and RAG systems
- DevOps and infrastructure for applied AI

## Site Sections

- `/ai/` — AI systems, evaluation, and agent engineering
- `/llm/` — LLM workflows, prompting, and tool use
- `/devops/` — infrastructure, deployment, and reliability
- `/projects/` — selected builds and open work

## Local Development

```powershell
hugo server
```

## Build

```powershell
hugo --gc --minify
```

## Deploy

Netlify builds the site using [netlify.toml](netlify.toml).

## Suggested GitHub Profile Organization

To mirror a strong technical profile presence:

- Use a concise profile bio centered on AI systems and infrastructure
- Pin 4-6 repos that cover writing, agents, retrieval, and infra
- Keep repo names concrete and domain-specific
- Maintain clean READMEs with problem, approach, and status
- Link the website, GitHub, LinkedIn, and RSS consistently
